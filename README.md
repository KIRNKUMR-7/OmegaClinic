# Omega Clinic

Welcome to the Omega Clinic website repository.

## Overview
This repository contains the codebase for the Omega Clinic website, featuring Homoeopathy, Alternative Medicine, and the Omni Being platform.

## Features
- **Omega Clinic**: General information, booking, and medical services.
- **Omni Being**: Spiritual wellness, Prana, Yoga, and Diet plans.

## Tech Stack
- Next.js
- Tailwind CSS
- Vercel

## Deployment
Deployed via Vercel.
